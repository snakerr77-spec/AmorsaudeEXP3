Amor Saúde Funcionários - versão corrigida

Alterações desta versão:
- Topo principal mais compacto e profissional.
- Seleção de cidade adicionada no rodapé do menu do perfil.
- Seleção de cidade também destacada dentro da página Meu Perfil.
- Cidades disponíveis: Embu das Artes, Itapeva, Tatui e Cerquilho.
- Nova área Controladoria com visual mais profissional.
- Controladoria liberada apenas para usuários com nivelAcesso admin ou financeiro.
- Controladoria com aparência de drive interno: pastas, documentos, pesquisa, editor, duplicar e excluir.
- Documentos iniciais: termos de responsabilidade, tabela de valores médicos, checklist financeiro, procedimento interno e pendências por unidade.

Como publicar no GitHub Pages:
1. Extraia este ZIP.
2. Envie/substitua os arquivos no repositório do GitHub.
3. Faça commit das alterações.
4. Confira se o Pages aponta para a branch correta.

Observação:
Os documentos da Controladoria ficam salvos no navegador pelo localStorage. Para todos os funcionários verem os mesmos documentos em tempo real, o próximo passo é ligar essa área no Firebase/Firestore.
